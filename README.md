# Lyrico

安卓平台的音频元数据编辑器与逐字歌词搜索应用，随缘更新...

## 功能特性

- **音频元数据编辑**: 修改本地音乐文件的标题、艺术家、专辑等信息。
- **歌词搜索**:
    - 支持从 **酷狗音乐** 和 **QQ音乐** 搜索歌曲信息。
- **逐字歌词**:
    - 目前支持 KRC (酷狗音乐) 和 QRC (QQ音乐) 格式的逐字歌词。
